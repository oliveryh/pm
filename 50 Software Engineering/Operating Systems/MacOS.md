# Shortcuts

- Toggle Full-screen Mode: <kbd>Command</kbd> + <kbd>Control</kbd> + <kbd>F</kbd>
- Bring up Spotlight: <kbd>Command</kbd> + <kbd>Space</kbd>
- Switch tabs in the same application: <kbd>Command</kbd> + <kbd>`</kbd> (Key above tab)
- Update `Move to Next Window` to Command + \ to be used with keyboard
