# Setup

## Prettier

Prettier is an opinionated formatter for code, we can format on save by doing the following:

1. Go to **File | Preferences | Settings**
2. Searching for **formatOnSave**
3. Enabling this setting

## Python Tests

1. Install the Python Extension
2. Setup the interpreter and tests directory by clicking on **Ctrl + Shift + P** and clicking on
	`python: Select interpreter` and `python: Configure tests` respectively
3. Install the extension Python Test Explorer
