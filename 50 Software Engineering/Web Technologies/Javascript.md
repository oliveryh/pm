# Overview

The key key technologies and trends to aim for in JS are as follows

- **Front End** - [[React]] ([[Vue]] close second)
- **Flavour** - [[TypeScript]]
- **Data Layer** - [[GraphQL]] (Redux getting bad response)
- **Backend** - [[ExpressJS]] (Next.JS close second for server side rendering)
- **Testing** - Jest and Storybook
- **Mobile** - Neutral on all options

Consider the following as important technologies to follow:

- Promises
- ES6
- Progressive Web Apps
