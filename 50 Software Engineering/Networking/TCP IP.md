# Layers

- Application (Website)
- Transport Layer (TCP / UDP)

Application communicates to the Transport layer by a port.

1. Creates packets from data
2. Adds headers to indicate order of packets
3. Sent to internet layer with destination and origin addresses via Internet Layer with (IP)
