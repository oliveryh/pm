# Bringing Value

# Professional Skills

## Communication

- [NoHello](http://www.nohello.com/) - Don't start a message on chat with just 'Hello'. Explain why you are contacting them in the first message so they can respond.

## Learning from Mistakes

### Asking for Help

1. Clearly state what the problem is
2. Ensure I've tried something myself, clearly stating my attempts and describe my current situation
3. Outline potential future choices and what the consequences/details of those are.
