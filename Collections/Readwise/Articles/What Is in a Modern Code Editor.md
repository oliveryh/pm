## Highlights
- We have [Tree-sitter](https://blog.meain.io/2022/modern-text-editor/tree-sitter.github.io/) which unlike using regexes actually parses your code and forms a syntax tree which you can query on top to figure out how to highlight your code. Tree-sitter is useful for a lot more than just syntax highlighting, but it makes syntax highlighting a lot more efficient.
