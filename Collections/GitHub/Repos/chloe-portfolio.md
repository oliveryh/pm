# Hugo vs Astro

- Install Hugo ([getting started guide](https://gohugo.io/installation/macos/))

I started the project with hugo because this is what I used to create my past blog. However, given the possibility of extending and amending functionality, I'd be far more comfortable working with javascript than golang.

# Initilisation

I started with a template [dante-astro-theme (JustGoodUI) (44 ★)](https://github.com/JustGoodUI/dante-astro-theme) and used the following starter command: `npm create astro@latest -- --template JustGoodUI/dante-astro-theme`
