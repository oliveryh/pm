# Context
- We provide excel templates to users to upload their portfolio or take actions on the platform
- Noticed that manually updating and managing templates was a big time commitment for staff as we grew and was prone to human error, sometimes impacting customers

# Action
- Suggested automating the creation of these templates using openpyxl and configuring as code
- Prototyped and demonstrated this

# Result
- Resulted in fewer mistakes, an ability to make changes quickly, and in the future, integrate with a translation tool

# Learnings
- Process shouldn’t sit with a key person
